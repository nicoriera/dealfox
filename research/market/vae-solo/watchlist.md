# VAE Solo — Market Watchlist

## Monitoring objective

Track the French market for new, refurbished and recent used electric bikes matching the VAE Solo 2027 profile.

Only meaningful events should generate alerts:

- significant price decrease;
- exceptional refurbished or used offer;
- attractive financing;
- new model offering clearly better value;
- size availability on a high-priority model.

## Data to retain per offer

- model and exact trim;
- model year;
- frame size;
- condition;
- seller;
- current price;
- reference price;
- observed date;
- mileage for used/refurbished bikes;
- battery health information;
- warranty;
- financing;
- included accessories;
- delivery cost;
- evidence URL;
- confidence level.

## Alert policy

Do not alert merely because a product exists.

Alert only when the event is actionable and materially changes the decision.

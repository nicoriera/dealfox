# Moma E-Longtail — Lock and security notes

## Context

After buying the Moma E-Longtail reconditioned at 1,999 EUR, security became an immediate ownership concern.

The family considered the **Kryptonite Evolution Mini-7 with KryptoFlex cable**.

## Assessment

The Kryptonite Evolution Mini-7 is a strong first lock choice for this use case.

Positive signals:

- U-lock format;
- compact shape;
- Sold Secure Gold certification;
- ART2 certification;
- cable included for secondary protection;
- good value for money;
- suitable for daily stops such as school runs and groceries.

## Important limitation

The cable is not equivalent to a chain.

It should be used only to secure secondary elements:

- front wheel;
- saddle;
- child seat accessory;
- light accessories.

The U-lock must secure the frame to a fixed point.

## Recommended locking strategy

### Short stop

- U-lock through frame and fixed point.

### Medium stop

- U-lock through frame and fixed point;
- cable through front wheel or accessories.

### Long stop / beach / city centre

- U-lock;
- cable;
- remove the battery if possible;
- consider adding a chain later.

## Future upgrade

If the bike is frequently parked for several hours outside, add a second lock:

- chain lock;
- or grinder-resistant U-lock.

## DealFox learning

Security recommendations should not be based only on bike price.

They should depend on:

- bike value;
- parking duration;
- parking location;
- visibility;
- battery removability;
- insurance requirements;
- family routine.

DealFox should eventually recommend an ownership pack, not only a bike.
